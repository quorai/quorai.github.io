What is Quorai?

Quorai is an open-source, educational proof-of-concept for AI-driven investment analysis. A quorum of LLM agents — each embodying the investment philosophy of a legendary investor — debates ticker signals, applies risk constraints, and produces a final trading decision. No real trades are ever executed.

The name blends quorum (a deliberating body) with AI, reflecting the system's core mechanic: structured disagreement among agents before any conclusion is reached.

How it works

  1. 1
    Preflight — each trading day, the pipeline optionally classifies the current SPY market regime (bull / bear / risk-off / neutral) and loads per-agent conviction weights built from prior labeled runs.
  2. 2
    Analyst agents — 25 personas independently analyse each ticker using price history, fundamentals, insider trades, and news. When regime selection is active, only the strategy groups relevant to the current regime run (e.g. risk-off → macro, deep value, sentiment only).
  3. 3
    Debate node — agent votes are aggregated into group consensus signals. When conviction weights are loaded, each analyst's vote is scaled by their rolling directional hit-rate before the group sum.
  4. 4
    Risk manager — pure maths: volatility- and correlation-adjusted position limits are computed without an LLM call.
  5. 5
    Portfolio manager — LLM emits a final buy / sell / hold decision with sizing and reasoning. Per-agent signals are written to a JSONL log that feeds back into the next run's conviction weights via the forward-return labeler.

Analyst groups

Deep Value
Ben Graham, Michael Burry, Mohnish Pabrai, Joel Greenblatt — margin of safety, intrinsic value, contrarian bets.
Quality Compounders
Warren Buffett, Charlie Munger, Phil Fisher, Aswath Damodaran — durable moats and earnings quality.
Growth & Catalyst
Bill Ackman, Cathie Wood, Peter Lynch — high-conviction growth and activist-style catalysts.
Macro & Cycle
Ray Dalio, Stanley Druckenmiller, Howard Marks, Nassim Taleb, Rakesh Jhunjhunwala — macro regime, credit cycle, tail-risk.
Quant & Systematic
Jim Simons, Cliff Asness, Ed Seykota — momentum, factor models, systematic trend-following.
Sentiment & Analytical
Specialist fundamentals, valuation, news-sentiment, and market-sentiment analysts.

Run it yourself

CLI
uv run python src/main.py --tickers AAPL,MSFT
Backtest
uv run python -m src.backtesting --tickers AAPL,MSFT --model deepseek/deepseek-chat --model-provider OpenRouter
A/B compare
uv run python -m src.backtesting compare --mode both --tickers AAPL,MSFT --model deepseek/deepseek-chat --model-provider OpenRouter

Requires an API key for at least one LLM provider (OpenAI, Anthropic, Groq, …) and a Finnhub key for market data. See the README for full setup instructions.

⚠️ Educational & research use only. Quorai does not constitute investment advice. Agent signals are generated by language models and should not be used as the basis for real financial decisions. Past signals do not guarantee future performance.
