# quorai.github.io

Landing page for [Quorai](https://github.com/quorai/quorai), served at **[quorai.github.io](https://quorai.github.io/)**.

## What's here

Static site built with plain HTML, CSS, and vanilla JS — no build step required.

| File | Purpose |
|------|---------|
| `index.html` | Main page (About, Agents, Trading Flow tabs) |
| `style.css` | All styles |
| `app.js` | Tab switching, agent gallery, flow diagram rendering |
| `agents.json` | Agent data (name, group, philosophy, …) |
| `flow.json` | Trading flow graph data |
| `assets/` | Logo images |

## Editing

Just edit the files directly and push — GitHub Pages picks up changes on `main` automatically.

For content changes to the agent roster or flow diagram, edit `agents.json` / `flow.json`.

## App repository

The trading system itself lives at [github.com/quorai/quorai](https://github.com/quorai/quorai).
